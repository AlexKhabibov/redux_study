Страничка рецептов. кастомные рецепты, которые пользователь может сам собрать плюс доступ к публичным рецептам.

Информация по эндпоинтам:
* USDA FoodData Central — продукты, пищевая ценность, поиск по продуктам. Требуется бесплатный API-ключ data.gov.  
* TheMealDB — рецепты, ингредиенты, инструкции, изображения. Бесплатная версия доступна без регистрации через публичный API v1.  

Учебный проект использует бесплатный апи эндпоинт для следующих целей:
https://www.themealdb.com/api/json/v1/1/list.php?i=list - для получения списка продутов (ингредиентов) из которых будут собираться кастомные рецепты;
https://apis.io/apis/themealdb/themealdb/?utm_source=chatgpt.com - для доступа к публичным рецептам;

Цель проекта: Разобраться как работает RTK для управления данными приложения и createAsyncThunk для выполнения асинзронных задач, а менно для доступа к апи
